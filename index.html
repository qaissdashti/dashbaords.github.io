Analyze Arabic Positive and Negative comments from various industries

Why create such a project?

There is a large amount of freely available data on the web that can help many in making better decisions, and help other people lives better in the process.

The Whole Data Science/ML Stack

Using freely available data from twitter api, to capture tweets by handel, is nobody mentioning zain twitter handle, then capturing the tweet and running it into a arabic sentiment analyzer, to predict if the tweet is negative or positive. Then capture the number of negative or positive, stack them in a bar graph, then apply a plotly object to make them interactive.
